# Aurellys Cafe Digital Menu

Modern digital cafe menu website with responsive UI, category filter, realtime search, and WhatsApp ordering system.

## Features
- Modern dark cafe UI
- Responsive mobile layout
- Search menu system
- Category filter
- Best seller badge
- WhatsApp order integration
- Lightweight static web app
- GitHub Pages ready

## Technologies
- HTML5
- CSS3
- JavaScript
- GitHub Pages

## Project Purpose
This application was created as a lightweight digital menu solution for cafes and small businesses. Designed for fast access, modern branding, and easy customer interaction through WhatsApp ordering.

## Usage
Customers can:
- Browse menu items
- Search products
- Filter categories
- Order directly via WhatsApp
